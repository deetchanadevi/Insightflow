The PredictIQ Analytical Architecture

Transparent decision pipelines explaining ingestion, linear regression, seasonality scaling, and forecast confidence mapping.

1

1. Data Input Ingestion

Loads historic parameters from CSV uploads or native industry presets. Dates and values are indexed, outliers are smoothed, and timestamps are parsed.

  • CSV Schema Auto-Mapping
  • Missing Value Interpolation
  • Mean & Variance Normalization
2

2. Time-Series AI Engine

Decomposes time-series components into a linear growth trend combined with seasonal multipliers. Evaluates model standard error relative to raw values. This is a transparent, explainable time-series statistical model (OLS regression + seasonal decomposition), not a black-box model.

  • Ordinary Least Squares: \( y = mx + c \)
  • Seasonal Sine Wave Multipliers
  • Standard Error of Estimate (SEE)
3

3. Forecast Visualizer

Extrapolates calculated trend patterns into a 6-month forecasting timeline, bounded by a 95% confidence interval shaded band.

  • Trend Line Projection (Dashed)
  • Shaded Confidence Interval Band
  • Active Bar/Line Visualization
4

4. Prescriptive Decisions

AI Insight Panel generates contextual instructions based on upcoming slopes. Prescribes optimal order quantities, shift counts, and price hedges.

  • Safety Stock Buffer Calculations
  • Demand Outlier Risk Warnings
  • Prescriptive Strategic Options

Mathematical Foundations

Statistical Time-Series Model
\[\hat{Y}_t = \left(\beta_0 + \beta_1 t\right) \times S_t + \epsilon_t\]

Where \(\hat{Y}_t\) is the predicted value at index \(t\), \(\beta_0 + \beta_1 t\) represents the Linear Trend (OLS), \(S_t\) represents the Seasonal Index multiplier mapped to calendar periods, and \(\epsilon_t\) represents model residuals and random variations. The 95% Confidence Band is defined by \(\hat{Y}_t \pm 1.96 \times S_e\) (Standard Error of the model).
